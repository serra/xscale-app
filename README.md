# xscale-app
Desktop app for contributing on the XSCALE wiki.

Electron app that bundles the XSCALE wiki with ungit to deliver a smooth learning experience to XSCALE coaches.

 1. Download
 1. Hookup your github account
 1. Collaborate
 1. Share
